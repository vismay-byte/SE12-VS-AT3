# Software Engineering Project

**Project Overview**

This GitLab repository will contain all components of your Year 12 Software Engineering Project (Task 3).
You will design, develop, document, test, evaluate, and present an individual software solution.

Your teacher will assess your entire project directly from GitLab, so all files must be committed here.

# Project Report/ePortfolio

Your Project Report is a compilation of documents that you have completed and over the course of your Project Development. 
*Please see Task 3 on Canvas for more specific details.*

**1. Identifying and Defining**

Include:
- Problem being solved
- Description of your software
- Scope (what the system will and will not do)
- Target Audience
- Key features / characteristics
- Constraints (time, skills, tools, data, etc.)
- Assumptions about users or the scenario

<br>

**2. Researching and Planning**

Include:
- Software Development Approach & Justification
- Gantt Chart (Dependencies Included)
- Two Social or Ethical Issues
- Communication Plan with stakeholder
- Quality Assurance Checklist *(See Template)*
- Data Dictionary
- System Model

<br>

**3. Producing and Implementing**

Include:
- Flowchart 
- How security is implemented within the code. 
- Screenshots/Code Snippets to support. 

<br>

**4. Testing and Evaluating**

Include:
- Test Plan & Report *(See Template)*
- Feedback Summary
- Software Evaluation *(See Template)*

<br>

# Software Solution/Code Requirements

Your final program must show:

- Modular design
- Clear and user-friendly UI
- Meaningful variable and function names
- Code optimisation where appropriate
- Effective commenting
- Use of white space and readability
- Security practices

# Final Presentation

**Requirements:**

The presentation should be approx 4 minutes in length and the mp4 file should be a max of 100 MB. (This is to allow for it to be uploaded to GitLab)

<br>
The Presentation should include:

- Software Demonstration
- Professional Style and Tone
- Explanation of features
- Challenges/Solutions
- Benefits
- Answers to the following Questions:

 1. You created a Gantt chart that spanned 15 weeks - did you follow this plan accurately? How would you adjust your planning in the future? (You will not be penalised if your process was different to what actually occurred.)

2. In regard to the software solution and coding - what is your general feedback on how you think you went putting it together and testing it? Would you do anything differently if you had your time again? Explain your answer. 

3. What advice would you give a future student about undertaking this project?
