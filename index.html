Student Flight Planner

Training Flight Preparation

Build a route, generate a full navigation log, and check it against your own personal minimums. This is a tool, built for RPL/PPL students training around Bankstown, Camden and the Sydney Basin. This has been designed to focus on the core navigation skills, not to replace official charts or weather sources. You will be able to develop charts and track your flights.

                  NAVLOG

                  Navigation log

                  Assembles the route built on the Route Planner page with a true airspeed, wind, departure time and aircraft into one navigation log, then lets you export it as a PDF. Uses the same wind-triangle, time/ETA and fuel engines as the preview panels there. This page has its own copy of those inputs so the printed log reflects one clear, final set of figures rather than whatever was last left in a preview panel.

                  Always cross-check against current ERSA/AIP/NOTAMs, certified aviation weather and the aircraft's actual POH before any real flight.

                  # Leg Dist TC WCA TH MH GS ETE ETA Fuel used Fuel rem.

                  Plays back the NAVLOG generated above against a simulated clock, with spoken (and logged) reminders for fuel checks, DI/compass checks, leg changes and checklist prompts, a training aid for practising the flow of a flight, not a real cockpit timer. Generate a valid log above first.

                  This browser doesn't support spoken callouts (the Web Speech API) and the timed log below will still run and display each callout as text.

                      Go / No-Go

                      Personal minimums and Go/No-Go

                      Enter your own personal limits once (saved on this device), fetch indicative current weather for the route's departure aerodrome, and get a plain Go / Caution / No-Go check with the specific reasons why.

                      Weather is indicative (Open-Meteo), not certified aviation weather. Always make the actual go/no-go call yourself using current ERSA/AIP/NOTAMs, a certified weather briefing and your own judgement.
